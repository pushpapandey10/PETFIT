# PETFIT
PetFit is a user-friendly web platform designed to help pet owners find nearby vet clinics, pet stores, and essential pet care resources. The website provides an intuitive and engaging experience for users to ensure their pets receive the best care and services available.

# Features
Find Nearby Vet Clinics – Locate the nearest vet clinics with ease.

Find Nearby Pet Stores – Discover pet stores in your area for pet essentials.

GPS Tracking – Track your pet's live location for safety.

Health Monitoring – Stay updated on your pet’s health and wellness.

Blog Section – Get expert tips and pet care advice.

User-Friendly Interface – Responsive and easy-to-use design.

# Technologies Used
Frontend: HTML, CSS, JavaScript

API Integration: Leaflet API
